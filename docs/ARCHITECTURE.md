## Component Responsibilities

### Frontend (React)
- **Dashboard:** Display real-time metrics and fraud alerts
- **Transactions:** Search, filter, and view transaction details
- **Drift:** Monitor data/model drift reports
- **Investigation:** Review and provide feedback on frauds
- **WebSocket Client:** Receive real-time updates
- **API Client:** Make REST API calls

### Backend (Node.js/Express)
- **Webhook Endpoint:** Receive predictions from pipeline
- **REST API:** Serve data to frontend
- **WebSocket Server:** Broadcast real-time updates
- **Database Layer:** Query PostgreSQL
- **Business Logic:** Process requests, format responses
- **Error Handling:** Catch and log errors

### Database (PostgreSQL)
- **Storage:** Persist all transaction and prediction data
- **Querying:** Efficient data retrieval with indexes
- **Relationships:** Foreign keys between tables
- **Analytics:** Aggregations for metrics
- **History:** Track changes over time
